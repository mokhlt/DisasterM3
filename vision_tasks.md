# Basic Computer Vision Tasks

Computer vision is about helping computers understand what is inside images.

## Classification

Classification means the model gives one label to the whole image.

For example, a satellite image can be classified as flood, wildfire, damaged area, or normal area.

This is useful when we only need a quick understanding of what is happening in the image.

## Detection

Detection means the model finds objects inside the image and shows where they are.

For example, in a disaster image, the model can detect damaged buildings, blocked roads, cars, or shelters.

This is more detailed than classification because it gives the location of the objects.

## Segmentation

Segmentation means the model separates the image into different regions.

For example, in a flood image, segmentation can mark water areas, roads, buildings, and safe land.

This is useful when we need exact details, like how much area is flooded or which parts of a city are damaged.

## Simple Difference

Classification tells what the whole image shows.

Detection tells what objects are inside the image and where they are.

Segmentation tells the exact areas or pixels that belong to each class.
